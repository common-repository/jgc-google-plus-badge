A simple plugin that creates a widget to display the Google+ Badge.

== Description ==

This plugin creates a widget to display Google+ badge. Just paste your Google Plus URL (profile, page or community) on widget options. Then, if you want, changes width, color scheme and layout (portrait or landscape).

== Installation ==

For an automatic installation through WordPress:

1. Go to Plugins > Add New
2. Search for 'JGC Google Plus Badge'
3. Click 'Install Now' and activate the plugin
4. Go to Appearance > Widgets

For a manual installation via FTP:

1. Unzip the file 'jgc-google-plus-badge.zip'
2. Upload the 'jgc-google-plus-badge' folder to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to Appearance > Widgets

To upload the plugin through WordPress, instead of FTP:

1. Go to Plugins > Add New
2. Click 'Upload Plugin' button
3. Select 'jgc-google-plus-badge.zip' file
4. Click 'Install Now' and activate the plugin
5. Go to Appearance > Widgets
